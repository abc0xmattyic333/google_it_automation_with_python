# Implicit Conversion

The interpreter automatically converts one data type into another.

```
print(7+8.5)
# output 15.5

print("a"+"b"+"c")
# output abc

print("This ", "is ", "pretty ", "neat! ")
# output This is pretty neat!
```

```
base = 6
height = 3
area = (base*height)/2
print("The area of the triangle is: " + str(area))
# output The area of the triangle is: 9.0
```
