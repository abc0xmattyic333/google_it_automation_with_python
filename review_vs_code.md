# Review: Use VS Code

VS code is an open-source editor that includes developer tooling. It is similar to Jupyter Notebooks and Colab, but includes more features.

To use Python, VS Code has an extension available. On Windows you can hit (Ctrl+Shift+X) and it will open the extensions marketplace. Then type in Python.

I have made this entire tutorial using VS Code. Thanks [VS Code](https://code.visualstudio.com/) for making this possible!<3 :)
